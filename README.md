# Market Risk Analysis Project

This project simulates a real-world market risk analysis workflow
used in global investment banks.

## What you'll learn
- How to analyze commodity market risk using SQL
- How to calculate portfolio exposure
- How to perform simple risk scenarios

## Project Structure
- data/: synthetic commodity data
- sql/: SQL queries for market risk analysis

## How to use
1. Download this repository as a ZIP file
2. Open the `sql/` folder
3. Run queries using your preferred SQL tool
